# 台灣車牌辨識模組技術文件 (Taiwan License Plate Detection)

## 1. 專案總覽 (Overview)
本資料夾 (`TaiwanLicensePlate`) 包含針對台灣車牌進行偵測的相關腳本與訓練模型結果。為尋找最適合的辨識模型，本專案累計測試了 5 個版本的模型，包含不同解析度與任務類型的比較：**YOLO V4**、**YOLO V5 (960 解析度)**、**YOLO V5 (1280 解析度)** 以及最新的 **YOLO V5 (1280) 語意分割 (Segmentation)** 模型。

資料夾主要包含以下子目錄：
* `CV2/`: 車牌影像前處理與特徵提取相關腳本。
* `YOLO/`: 包含檢測模型的測試結果與效能評估數據，用於進行模型的比較與選擇。

---

## 2. 模型評估與比較架構 (Architecture)

以下為本車牌辨識模組的資料處理與模型評估流程：

![模型評估架構](architecture.png)

1. **台灣車牌資料集**：輸入經標記的台灣車牌圖庫。
2. **模型訓練**：將相同資料集分別投入 YOLO V4 與 YOLO V5 (含不同解析度) 進行訓練。
3. **效能指標評估**：針對各模型最佳權重 (`best.pt`)，進行 Precision、Recall、mAP50、mAP50-95 與 F1 Score 的評估。
4. **模型比較與結論**：整合並比較所有模型數據，挑選表現最優異的模型作為最終應用。

## 4. 實驗結果與表現比較 (Performance Comparison)

為確保比較的公平性，我們針對每個模型抓取了使其 Fitness (綜合評分) 達到最高的 Best Epoch 數值進行對比。

| 模型名稱 | 最佳 Epoch | Precision (精確度) | Recall (召回率) | mAP50 | mAP50-95 | F1 Score |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| **YOLO V4** | 121 | 91.78% | 91.67% | 94.97% | 63.09% | 91.72% |
| **YOLO V5 (960) Run 1** | 178 | 91.64% | 95.76% | 97.04% | 86.53% | 93.65% |
| **YOLO V5 (960) Run 2** | 180 | 91.16% | **97.14%** | 96.46% | 82.56% | 94.13% |
| **YOLO V5 (1280)** | 181 | **95.10%** | 96.95% | 97.76% | 86.05% | **95.91%** |
| **YOLO V5 (1280) Seg** | 200 | 94.86% | 96.86% | **98.43%** | **90.02%** | 95.84% |

*(註：F1 Score 係基於表格中各模型的 Precision 與 Recall 計算而得：`2 * (P * R) / (P + R)`)*

## 5. 分析與結論 (Conclusion)

1. **YOLO V4 vs YOLO V5**：
   在整體表現上，YOLO V5 系列在各項指標皆顯著優於 YOLO V4。特別是在嚴格的 `mAP50-95` 評估標準下，YOLO V4 的表現大幅下滑至 63.09%，而 YOLO V5 各版本皆能維持在 82% 以上，顯示 YOLO V5 在邊界框定位的精準度上有巨大優勢。

2. **偵測模型 (Detection) 表現比較**：
   * **YOLO V5 (1280)** 在 **Precision (95.10%)** 與 **F1 Score (95.91%)** 取得了偵測模型中的最佳成績，適合追求極致精確度的場景。
   * **YOLO V5 (960) Run 2** 則在 **Recall (97.14%)** 上表現亮眼，有效減少了漏判的情況。

3. **語意分割模型 (Segmentation) 的突破**：
   * **YOLO V5 (1280) Seg** 雖然在 F1 Score 上與偵測版持平，但在 **mAP50 (98.43%)** 與 **mAP50-95 (90.02%)** 取得了突破性的進展。
   * 高達 90% 以上的 `mAP50-95` 代表該模型在複雜角度、光照或部分遮擋下，對於車牌形狀的貼合度極高，能提供更穩定的邊界定位。

**最終建議**：
基於最新數據與綜合表現，推薦優先選用 **YOLO V5 (1280) Segment** 作為核心辨識模型。其優異的 `mAP50-95` 指標代表了極高的定位穩定度，配合語意分割技術，能更精準地提取車牌特徵，是目前 5 個版本中最理想的選擇。
